Liferay Faces is an umbrella project that provides support for the JavaServerTM Faces (JSF) standard within Liferay Portal.

The Liferay Faces Portal sub-project includes JSF utilities, EL keywords, and a suite of JSF UI components for use with Liferay Portal.

The Facelet component tags are prefixed with the portal: namespace are only compatible with Liferay portlet environments.
